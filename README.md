## Développement d'une application de location de logements ##

Ce projet avait pour objectif d'implémenter le front-end d'une application de location de logements en utilisant le framework React et Sass.

Principales fonctionnalités développées :

-> Initialisation d'une application avec Vite

-> Configuration de la navigation entre les pages avec React Router

-> Développement des éléments de l'interface avec des composants React

-> Mise en oeuvre d'animations CSS

-> Développement de l'interface avec Sass
